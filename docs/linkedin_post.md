🚀 Excited to share my latest project: the **Feedback Intelligence System**! 🔮

Understanding customer sentiment at scale is a challenge for modern product teams. To solve this, I built an end-to-end platform that automates the feedback loop—from ingestion to executive insights.

### 💡 What does it do?
1. **Real-time Ingestion**: Automatically analyzes customer reviews for sentiment (positive/negative/neutral) and categorizes tickets (bug, feature request, general, complaint) instantly.
2. **Prioritization Engine**: Uses NLP rules to assign an intelligent priority score (1-5) based on severity keywords and categories.
3. **Executive Dashboard**: Clean, Linear-inspired dark mode UI visualizing daily trends, sentiment distributions, and category counts using Chart.js.
4. **Gemini AI Integration**: Uses Google Gemini 1.5 Flash to generate executive summaries, root causes, and interactive checklists for product teams.
5. **System Health Checkers**: Inbuilt health tags monitoring backend, database, and LLM API status.

### 🛠️ Tech Stack:
- **Frontend**: React, Vite, Tailwind CSS, Chart.js, Axios, React Router.
- **Backend**: FastAPI, Python, TextBlob, SQLAlchemy.
- **Database**: SQLite & PostgreSQL.
- **Deployment**: Docker, Docker Compose, ready for Railway and Vercel.
