### **About / Subtitle**
🔮 AI-powered customer feedback intelligence platform. Automatically ingests customer reviews, classifies sentiment & categories, detects anomalies, and generates executive summaries and actionable checklists using FastAPI and Google Gemini 1.5 Flash. Styled with a Vercel-inspired React dashboard.

### **Repository Features Summary**
- **Real-Time NLP Processing**: Instantly scores sentiment polarity and tags categories (bugs, complaints, praise) upon ingestion.
- **Urgency Classification**: Heuristically ranks feedback with priority indices (1-5) depending on severity keywords and categories.
- **Interactive Vercel-inspired Dashboard**: Beautiful dark-mode charts showing timeline trends, distributions, and keyword progress bars.
- **Gemini AI Command Center**: Produces high-level business summaries, root causes, and interactive action recommendation lists.
- **Live Health Monitoring**: Pulsing status indicators monitoring Backend Server, SQLite Database, and Gemini AI state.
- **Seeding Pipeline**: Built-in demo seeding tool to simulate stream ingestion.
- **Dockerized Environment**: Ready-to-go multi-container configurations.
