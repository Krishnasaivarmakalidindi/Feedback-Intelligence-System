### **Lead Full Stack & AI Engineer | Feedback Intelligence System**
*Developed a real-time, AI-driven customer feedback analysis and intelligence platform processing high-volume text streams to extract actionable insights.*

- **System Design & API Orchestration**: Architected and built a decoupled Single Page Application (SPA) in **React (Vite, Tailwind CSS)** connected to a **FastAPI** backend endpoint serving CRUD operations, caching pipelines, and dynamic analytics.
- **NLP & Sentiment Classification**: Implemented local NLP analysis pipelines utilizing **TextBlob** to perform real-time sentiment scoring (positive/negative/neutral) and automatic category detection (bugs, feature requests, complaints) during ingestion.
- **Google Gemini AI Integration**: Integrated the **Google Gemini Pro API** (1.5 Flash) on the server tier to generate executive summaries, root cause breakdowns, and prioritized checklists based on rolling analytics historical windows.
- **Data Visualizations**: Designed interactive charting boards with **Chart.js** displaying volume timelines, category distributions, sentiment doughnut charts, and word keyword progress meters.
- **System Health & Monitoring**: Designed operational monitoring interfaces with dynamic health status checkers for the API server, database connection pools, and Gemini services, handling missing credentials gracefully.
- **Operational Efficiency**: Configured **Docker & Docker Compose** layers to orchestrate database migrations, backend workers, and static SPA servers, achieving containerization-ready configurations.
